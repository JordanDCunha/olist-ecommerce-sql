# 🛒 Olist E-Commerce SQL Analytics Project

## 📌 Overview
This project builds a full PostgreSQL data pipeline using the Brazilian Olist e-commerce dataset (100K+ orders).

It includes:
- Database schema design
- Data loading from CSV files
- Data transformation and feature engineering
- Business-focused SQL analytics

---

## 🏗️ Project Structure

- `schema.sql` → Database tables and relationships
- `data_load.sql` → Load raw CSV data
- `transform.sql` → Clean and prepare analytics-ready datasets
- `analytics.sql` → Business insights and queries

---

## 📊 Key Analyses

- Monthly revenue trends
- Customer lifetime value (CLV)
- Top-selling product categories
- Delivery performance analysis
- Customer segmentation (repeat vs one-time)
- Cohort analysis

---

## 🧠 SQL Skills Demonstrated

- Complex JOINs
- Aggregations & GROUP BY
- Window functions (RANK, running totals)
- CTE-style logic via views
- Date/time analysis
- Data modeling & transformation

---

## 📁 Dataset

Source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Place all CSV files in a `/data` folder before running scripts.

---

## 🚀 How to Run

1. Run `schema.sql`
2. Run `data_load.sql`
3. Run `transform.sql`
4. Run `analytics.sql`

---

## 💼 Resume Bullet

Designed and implemented a PostgreSQL data pipeline analyzing 100K+ e-commerce transactions; built relational schema, performed data transformation, and generated business insights using advanced SQL (joins, window functions, cohort analysis).
